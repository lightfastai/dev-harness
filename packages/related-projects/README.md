# @lightfastai/related-projects

Development helpers for Lightfast related projects using Vercel microfrontends and portless.

## Usage

```ts
import { withPortlessMfeDev } from "@lightfastai/related-projects/next";

export default withPortlessMfeDev(nextConfig);
```

The package also provides the `portless-mfe` CLI for running local microfrontend dev commands through portless.
